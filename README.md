## 프로젝트 소개

이 프로젝트는 ARP 패킷을 보내 상대방 IP의 인터넷 접속을 방해하는 프로젝트입니다. 

* 내부망 IP 검색 기능
* ARP 차단 기능
* 내 내부 IP 자동 기입 기능

## 프로젝트 개발 동기

NetCut이라는 소프트웨어가 있습니다. 이것도 마찬가지로 인터넷 접속을 방해하는 소프트웨어인데, 부분적으로 유료화 되어있습니다. 더 많은 인터넷 사용자를 차단하기 위해서는 비용을 지불해야 합니다. 이 부분을 해결하고자 오픈소스로 개발했습니다.
